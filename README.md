# HTML & CSS para iniciantes

Mini-curso @ [SECCOMP 2015](http://seccomp-unesp.github.io)

### CONTEÚDO
HTML
- história
- elementos básicos
- alguns exemplos

CSS
- classes, identificadores e encadeamentos
- ordem e pesos na declaração
- padding e margin
- unidades de medida
- displays e float
- positions e z-index
- prefixos de extensão
