##  CSS - Identificadores

```css
#pokemon {
  font-size: 90px;
  font-weight: bold;
  color: #ffcc00;
  border: solid 5px white;
  background-color: rgb(109, 156, 190);
  padding: 30px 20px;
}

```

> Identificadores são utilizados quando o estilo é único e so deve ser utilizado em apenas um elemento.
