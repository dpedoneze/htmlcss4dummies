##  CSS - Classes

```css
.pokemons {
  font-size: 36px;
  padding: 10px;
  border: solid 1px white;
  color: #fff;
}

.water-type {
  background-color: rgb(109, 156, 190);
}

.grass-type {
  background-color: rgba(109, 156, 54, 1);
}

.fire-type {
  background-color: hsl(25, 89%, 50%);
}
```

> Classes não são únicas e por esse motívo podem (e devem) ser reutilizadas
