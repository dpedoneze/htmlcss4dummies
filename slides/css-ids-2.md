##  CSS - Identificadores

Exemplo do identificador `#pokemon` sendo aplicado em uma `div`

<style type="text/css">
#pokemon {
  font-size: 90px;
  font-weight: bold;
  color: #ffcc00;
  border: solid 5px white;
  background-color: rgb(109, 156, 190);
  padding: 30px 20px;
}
</style>

<div id="pokemon">POKEMON!</div>
